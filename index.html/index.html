Objetivo general: Realizar un diseño de un sitio web mediante varias herramientas y lenguajes de programación, elaborando un sitio seguro estable y al cual puedan trabajar varias personas desde diferentes puntos del mundo sin ningún inconveniente. Y mejorar las habilidades con las distintas herramientas tecnológicas que nos ofrece el mercado. Objetivo específico 1: mediante la herramienta de desarrollo de software GITHUB desarrollar un sitio web colaborativo, donde varias personas puedan trabajar al mismo tiempo sin problema y la herramienta les ayude a integrar su desarrollo sin temor a dañar los avances de sus compañeros. De esta manera ser mas agiles y efectivos a la hora de crear un sitio web en grupo Objetivo específico 2: Manejar y utilizar los lenguajes de programación de html 5, (estructuras, valores, etiquetas, atributos) contenidos en HTML 5 (BODY, HEAD, SECTION, FOOTER, ARTICLE) manejar estilos internos y externos con selectores en css y css3. Solo se manejaran ests lenguajes ningún otro. Objetivo específico 3: realizar navegación interna y externa con ( NAV – UL – OL –LI) múltiples clases float, position, oweflow. Una maquetación del sitio web paa presentación al cliente y no desgastar el equipo de trabajo. Permitiendo a los estudiantes navegar e investigar sobre cada uno de los contenidos estipulados en el curso. Y acceder a diferentes medios audiovisuales en la pagina web
